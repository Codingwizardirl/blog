Linux is great

but...

Posted by Pavel Georgiev on 2016-08-23

I have been running Windows on my machines as long as I can remember. I have been through XP, Vista, Widnows 7, Windows 8, and currently am running Windows 10 in dual boot with Linux, but more about that later. And if we pretend that Vista never existed, for the most part, I was happy with my experience.
However, during my internship, I began using my machine primarily for web development and this is where I began hitting road blocks on daily basis. Turns out that having to deal with web developer tools on Windows is a complete nightmare! I just wanted to focus on the work I had instead of jumping through million hoops just to install the dependencies for a NPM package.

Previous to that I have been on the verge of dual booting Linux distribution alongside my Windows 10 for a long time. What was putting me of was that I had to deal with all the hassle connected with installing another OS(disk defragmentation,UEFI vs EFI boot modes and etc.). Luckily for me, Windows had just recently announced their Windows Subsystem for Linux project, which potentially meant I could still skip installing a second OS. Because I could easily run Ubuntu bash on Windows 10, I was hopping it will solve my troubles and decided to give it a try. Suddenly every problem became a lot more enjoyable to deal with. For a person who had never touched Linux operating system by then the “apt-get” package manager seemed like a godsent. At this stage I had a “real” terminal, which coupled with ConEmu made the next few weeks of my development flow pretty enjoyable. I was still running into problems, though. A lot less of them now, but because they were specific to the WSL(Windows Subsystem for Linux) project there were still no solutions for most of them. So… another road block.

At this point I had little to no choice. I configured my Lenovo Y50 laptop for dual booting(which turned out far less troublesome than I expected) and in a day or so I had a brand spanking new Ubuntu Gnome 16.04 LTS. It felt like everything was a terminal command away. And because of me being me and Linux being extremelly customizeable, I spend a solid amount of time just making sure my machine looks pretty. I loved it! Development-wise I have absolutely no complaints so far and have been using it daily for almost 3 months now.

Pretty. Prety. Shiny. Shiny!

Sadly, it is not only rainbows and sunshines. Media consumption on Ubuntu 16.04 is a whole different beast. The Nvidia Optimus configuration on my laptop just doesn’t work. I tried everything available from Bumblebee to Nvidia Primus and nothing helped. After a week struggle to use the proprietary Nvidia drivers, which resulted in black screen after reboot I just quit and decided I don’t want my Nvidia card on Linux anyways.
I can’t really enjoy playing music through my speakers too, as the subwoofer on my laptop does not work on Ubuntu 16.04 for some reason. Again the fixes for previous Ubuntu releases do not fix it. On the bright side, I get amazing sound quality on healdphones because I could use Linux’s ALSA tool to tweak it.
Lastly, when dealing with dual monitor setup the situation isn’t that pretty either. Well, GNOME does a good job, as long as displays are not with a widely varying DPI. Try and guess in which boat I am… Because at this point I don’t have a 4k+ monitor and my laptop screen is UltraHD I ran into problems with scalling on the secondary monitor. However, with a bit of help from the Internet I fixed it and will be writting a short blog post on how to do that yourself in the upcomming week or so.

Overall, I fell in love with Ubuntu and am willing to try other distros in the near future if the hardware I have allows me to run things without (many) problems. Arch Linux I am looking at you. However, I am still not convinced I can switch to standalone Linux setup until the issues I mentioned are fixed or I just simply have hardware that would allow me to not run into them.

Oh boy my first real blog post and it turned out to be a huge wall of text, so only today, since I am feeling a bit generous, you get 3 potatoes for the price of one.
